# site-leyu-hub

这是一个用于归档和发布多个独立 HTML 页面的仓库。仓库内收录了若干静态网页文件，方便统一管理、版本追踪与在线访问。

## 项目简介

site-leyu-hub 主要承担以下功能：

- 存放不同用途的独立 HTML 页面
- 通过 GitHub Pages 进行静态页面发布
- 提供清晰的目录结构，便于查找和使用

本仓库不针对任何特定域名或网站，所有页面均为独立归档，不包含后台逻辑或动态内容。

## 目录结构

```
site-leyu-hub/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的资源文件（图片、样式、脚本等）
├── docs/           # 相关文档或说明
└── README.md       # 本文件
```

各目录可根据实际需要调整，新增页面建议放置在 `pages/` 目录下，资源文件统一存放于 `assets/` 中。

## 页面归档说明

- 每个 HTML 页面均为独立文件，可直接在浏览器中打开使用。
- 页面之间无强依赖关系，可单独维护和更新。
- 归档内容以静态页面为主，不涉及服务器端处理。
- 文件命名建议简洁明了，便于识别用途。

## 维护说明

- 欢迎提交 Pull Request 添加新的 HTML 页面或优化现有页面。
- 添加页面时请同时更新 `pages/` 目录下的索引文件（若有）。
- 请确保页面内容不违反平台相关政策，不包含恶意代码或非法信息。
- 资源文件请尽量压缩优化，减小仓库体积。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `pages/` 目录下的 HTML 文件。
3. 若启用 GitHub Pages，可通过仓库设置中的 Pages 功能在线访问。
